# Planeta_test_task
Video stream via pyzmq

Quick start:
```
git clone https://github.com/flild/Planeta_test_task.git
pip3 install -r Planeta_test_task/requirements.txt
```
Start server
```
python3 server.py
```

Start client
```
python3 client.py
```

When the client is connected, the video will be streamed to him
